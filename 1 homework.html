  • snake_case: my_first_project (underscores)
  • kebab-case: my-first-project (hyphens)

Common Mistakes to Avoid:

  • Forgetting to save files before running
  • Using spaces in project names
  • Not checking if tools are installed
  • Forgetting file extensions when running
